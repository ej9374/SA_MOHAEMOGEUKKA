# 😋 모해모그까

## 기획배경 및 설명

레시피에 대한 관심은 높지만 바쁜 일상 속에서 매번 새로운 요리를 탐색하는 데 어려움을 겪는 사용자들이 많습니다. 특히 재료 준비 및 메뉴 결정 과정이 번거롭게 느껴져 식사 선택이 제한되는 경우가 자주 발생합니다. 또한 개인의 취향이나 건강 관리 목적에 따라 **맞춤형 레시피 추천**에 대한 수요가 존재합니다.

- 사용자가 평가한 데이터를 기반으로 **개인화 레시피 추천 제공**
- 레시피 상세 화면에서 **유사 레시피 추천**을 통해 선택지 확장
- 신규 사용자에게 인기 레시피 평가를 유도하여 **Cold Start 문제 해소**

이러한 기능을 통해 사용자가 메뉴 고민 없이 취향에 맞는 음식을 발견하고, 다양한 레시피를 경험하며 식사 선택을 즐겁게 만들 수 있는 환경을 제공하고자 했습니다.

- 진행 기간: 2024.09. ~ 2024.12.(4개월)
- 팀 구성 및 담당 역할: 총 3명 (추천 알고리즘 2명, **프론트·백엔드 1명**)

## 나의 역할 및 구현 내용

- 회원가입 및 로그인 구현
- 사용자별 레시피 평점 수집 및 저장
- Flask 추천 모델 연동
- 레시피 데이터 전처리 및 DB 모델링
- 추천 결과 및 유사 레시피 조회 API 개발
